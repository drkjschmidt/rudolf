\subsection{Detailed Description}
Manipulate date stamps as used by NMEA / GPS. NMEA date is reported as an int32 value in human-\/readable dd-\/mm-\/yy format (010100 to 311299) ... considering the age of NMEA there should be a rollover from 1999 to 2000 and maybe another from 2099 to 2100 but since this code is written in 2009/2010 neither of these should bother us!
